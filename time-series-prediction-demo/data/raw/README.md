# Local Dataset CSV Files

Full dataset CSV files are not committed because they are too large.

Place files here before running default predictions:

```text
appliances_energy_dataset.csv
beijing_air_quality_dataset.csv
bitcoin_dataset.csv
hanoi_air_quality_dataset.csv
sunspots_dataset.csv
```

You can also leave this folder empty and upload a compatible CSV through `CSV Override` in the demo UI.
